# Petrol_Price_Analysis_Economics

## Project Overview
In this project I explored the macroeconomic drivers of global petrol prices. Using data analysis and econometric modeling (Ordinary Least Squares), it investigates the relationship between pump prices, global crude oil prices, inflation, and GDP growth. It features modular architecture, diagnostic testing for multicollinearity (VIF), and automated unit tests.

## Project Structure
* `src/`: Contains external Python modules for data preparation, visualization, and modeling.
* `tests/`: Contains unit tests to ensure the data pipelines function correctly.
* `environment.yml`: Contains the Conda environment dependencies required to run the code.
* `Final_PetrolPrices_Analysis.ipynb`: The main narrative notebook containing the EDA, regressions, and interpretations.
